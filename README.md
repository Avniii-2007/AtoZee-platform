# AtoZee-platform
My second project basically a E-Commerce platform
AtoZee is basically a mini E-commerce platform similar to Amazon, Flipkart. 
It will have features like product listing, search bar for searching products, category filter, product details, also it is responsive
I used "https://dummyjson.com/products" for fetching api
it has features like dark mode/ light mode, filter and search
